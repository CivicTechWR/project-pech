## About

PECH is a civic tech initiative by CivicTech Waterloo Region to address chronic homelessness. The platform provides:

- Data visualization of homelessness statistics
- Community spotlights and personal stories
- Information about the Lived Expertise Panel
- Updates on housing initiatives and programs

## Goal

Help Waterloo Region reach **functional zero chronic homelessness by 2030**.

## Tech Stack

- Next.js 15 + React 19 + TypeScript
- Tailwind CSS v4
- Directus CMS backend
- Deployed on Vercel

## Development

```bash
bun install
bun run dev
bun run build        # Build for production
bun run lint         # Run ESLint
bun run check-types  # TypeScript type checking
```
